# ROB599
`ROB599` is a ros2 package for the ROB599 HW2. This contains the groups world files and an implementation of the dynamic window approach (DWA).

## Installation
- Clone the git repository to the desired directory.
- Then open the directory and build in the root directory.

```bash
cd "Path/to/Directory"  
colcon build  
source install/setup.bash  
```  

## Dependencies
- ROS2 Humble + 
- Stage
- stage_ros2

## Usage
This repository contains ros2 packages for each hw of ROB599. Instructions for each package can be found in their respective README.
